# Systeme Solaire

Systeme solaire en OpenGL afin de se familiariser avec à l'échelle 1/100.

## Utilisation

Les touches Z,Q,S,D pour se déplacer dans le système solaire.
La touche A pour arrêter la rotation des planètes.
